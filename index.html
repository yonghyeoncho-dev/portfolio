UI 기반 설계 · 사내 솔루션

데브팩 — 21개 기업 UI의 공통 조상

이젠고 UI 개발 패키지의 앞단을 정립하다. 개별 프로젝트가 아니라, 다른 프로젝트들이 딛고 설 기반을 만든 작업입니다.

문제
매 프로젝트마다 UI를 처음부터 반복 제작 — 시간과 비용이 크고, 회사마다 화면 품질이 제각각이었습니다.
접근
대표가 구상한 UI 패키지 '데브팩'을 입사 직후 이어받아, 사내에 축적된 구축 사례에서 재사용 가능한 UI 패턴을 추출. 규칙 · 가이드 · 템플릿 · 디자인 등 UI 앞단을 사실상 전담해 표준 틀로 정립했습니다. (엔진 · 백엔드는 개발자와 협업)
확장
UI 규칙 · 컴포넌트 · 템플릿 체계를 디자인 가이드 문서(54p)로 표준화했고, WCAG 2.1 AA · KRDS(공공 UI 표준)를 반영해 접근성 기준까지 정의했습니다. PC에서 모바일 · 테마로 적용 범위를 넓혔고, 현재는 자동화 툴로 진화 중입니다.
결과
기획 · 디자인 · 개발 · 운영 전 직군이 따르는 공통 기준이 됐고, 이 UI 기반이 한전 · 한수원 · 두산건설 등 다수 기업 시스템에 재사용되며 개발 생산성 향상과 유지보수 비용 절감으로 이어졌습니다.

UI컨버전 자동화 솔루션

데브팩으로 세운 UI 기반을, 이번엔 사람이 아니라 도구가 찍어내게. 만드는 것에서 만드는 방식을 만드는 것으로.

문제
넥사크로 UI 컨버전을 매 건 수작업으로 처리 — 반복이 많고 시간 · 비용이 컸습니다.
접근
데브팩 기반 위에 컨버전을 자동화하는 솔루션을 설계 · 리딩. 스크립트 자동화로 반복 공정을 걷어냈습니다.
확장
솔루션에서 그치지 않고, 제품을 알리는 홍보 · 웹사이트 개편 · 문서 · 전체 틀까지 직접 설계했습니다. UI를 만드는 것을 넘어, 그 UI 제품을 시장에 내보내는 일까지.
